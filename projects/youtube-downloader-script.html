YouTube Video Downloader Powershell Script

This is a script for use with Windows PowerShell that can download YouTube videos for free.

I. Requirements

  • Windows 10/11
  • Windows PowerShell (Should be installed by default. If you do not have PowerShell, download it by following the guide from Microsoft )
  • The latest version of yt-dlp for Windows (download yt-dlp.exe)
    • Make sure to download all the dependencies for yt-dlp.

II. Where To Download

Download it from the latest release .

III. Installation and Setup

  1. Place the PowerShell script wherever you’d like to keep it.
  2. Place yt-dlp.exe wherever you’d like to keep it.
  3. Right-click the script and click Edit in Notepad.
  4. Go to line 73, which contains:
    $ytDlpPath = "C:\Path\To\yt-dlp.exe" # Update this!
  5. Replace C:\Path\To\yt-dlp.exe with the correct path to your yt-dlp.exe.

IV. Using The Script

  1. Right-click the script and click Run in PowerShell.
    • Alternatively, you can use a batch script for easier execution. See Using A Batch Script .
  2. A window will open asking for the video URL. Paste it in.
  3. Click OK.
  4. The window will show a progress bar. When it closes, your video is downloaded to C:\Users\your-user\Videos\Downloaded Videos.
